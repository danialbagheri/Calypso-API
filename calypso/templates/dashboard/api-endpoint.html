API Endpoints

Below are the list of Calypso API endpoints for front end development. There will be CORS security applied to all these endpoints.

Products Endpoints

This endpoint only allows "GET" requests and takes parameters below:
count, product, top, collection 

      
        /api/products/product/ #lists all products (max 30)

        /api/products/product/<product-slug>/ #query the database for the product by it's slug

        /api/products/product/?count=2 #limit the counts shown

        /api/products/product/?count=2&type=kids #product type 

        http://localhost:8000/api/products/product/?count=2&top=yes #filter top products 

        http://localhost:8000/api/products/product/?&collection=best #filter by collection

This endpoint is for product variants and only allows "GET" requests 

      
        /api/products/variant/ #lists all variants (max 30)

        /api/products/variant/<sku>/ #queries the database for the relevant sku
Reviews

Accepts "GET" and returns the approved reviews filtered by product slug 

        
            /api/reviews/product/<product-slug>/
Website

This endpoint only accepts "POST"
